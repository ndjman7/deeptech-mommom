# 인텔리시아 (Intellicia)

> 2025년 팁스 딥테크 트랙 선정기업 (패스트트랙)

## 기본 정보
- **분야**: 인공지능(AI) / 소비자 리서치
- **사업아이템**: AI 합성 소비자 생성 플랫폼 '더서베이(TheSurvey.ai)'
- **핵심 기술**: 대규모 언어 모델(LLM) 기반 합성 소비자(Synthetic Consumers) 생성 기술, 소규모 앙상블 모델

## 사업 내용
인텔리시아는 실제 소비자의 인구통계학적 정보, 행동, 취향 데이터를 기반으로 AI 응답자를 생성하여 빠른 시간 내에 대규모 설문조사를 수행하는 플랫폼 '더서베이'를 운영한다. 생성된 합성 소비자의 행동 분석을 통해 선호도, 인지도, 사용 패턴 등의 인사이트를 도출하며, 기존 소비자 조사를 대체할 수 있는 차세대 리서치 방식으로 평가받고 있다.

## 투자/성과
- 소풍벤처스로부터 5억원 규모 시드 투자 유치
- 2025년 딥테크 TIPS '패스트트랙' 선정 (1차 기술성 평가 고득점으로 2차 시장성 평가 면제)
- 총 17억원의 정부 및 민간 지원 확보 (연구개발비 15억 포함)
- 설립: 2023년 (본사: 대전광역시)
- 대표: 백승국 (데이블 창업자)

## 딥테크 포인트
합성 소비자 기술은 2024년 하반기부터 하버드대학교, 스탠포드대학교 등에서 연구가 본격화된 최신 AI 응용 분야이다. LLM 기반으로 실제 소비자 응답 데이터를 학습한 합성 소비자를 생성하고, 소규모 앙상블 모델을 활용하여 기존 설문조사의 비용과 시간 한계를 혁신적으로 줄이는 기술적 깊이를 보유하고 있다.
