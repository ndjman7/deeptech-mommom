# 슈타겐 (Shutagen)

> 2025년 팁스 딥테크 트랙 선정기업 (패스트트랙)

## 기본 정보
- **분야**: 제조 DX / AI 로봇 자동화
- **사업아이템**: AI 기반 로봇 자율제조 솔루션 '메타라인(MetaLine)'
- **핵심 기술**: 소프트웨어 정의 공장(SDF, Software Defined Factory) 플랫폼
- **대표**: 김원현
- **설립**: 2019년 2월

## 사업 내용
제조산업 디지털전환(DX) 전문 기업으로, 소프트웨어 정의 공장(SDF) 솔루션 '메타라인'을 개발한다. 메타라인은 파편화된 로봇 티칭 환경을 통합하고, AI 기반 최적화로 로봇 시스템의 생산성을 극대화하며, 디지털 트윈 환경에서 검증을 제공하는 차세대 제조 솔루션이다. 제조사마다 다른 로봇 시스템의 호환성을 높이고, 직관적인 작업 지시가 가능하도록 로봇 프로그래밍을 간소화하는 것이 핵심 목표이다.

## 투자/성과
- 딥테크 팁스 패스트트랙 선정 (2차 시장성 평가 면제)
- 3년간 최대 18억 원 R&D 자금 + 보육공간/멘토링/글로벌 진출 프로그램 지원
- 2023년 과학기술정보통신부 장관상 수상
- 스파크랩으로부터 시드 투자 유치 (2023년 12월)

## 딥테크 포인트
제조 로봇의 프로그래밍과 운영을 AI로 통합/자동화하는 것은 높은 기술적 난이도를 요구한다. 각 제조사별로 상이한 로봇 시스템의 호환성 문제를 소프트웨어 레이어에서 해결하고, AI 기반으로 로봇 제어 코드를 표준화하며, 디지털 트윈 환경에서 사전 검증하는 기술 스택이 핵심이다. 로봇 티칭의 자동화와 최적화는 제조업의 근본적 생산성 혁신을 가능하게 하는 딥테크 영역이다.
