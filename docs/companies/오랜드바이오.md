# 오랜드바이오

> 2025년 팁스 딥테크 트랙 선정기업 (패스트트랙)

## 기본 정보
- **분야**: 바이오 / 의료기기 / 조직재생
- **사업아이템**: 신장 기능회복 스캐폴드 (이식형 의료기기)
- **핵심 기술**: 생분해성 고분자 기반 생체적합 고성능 스캐폴드 기술
- **대표**: 한동근
- **설립**: 2024년 11월
- **추천 운영사**: 젠엑시스

## 사업 내용
생체재료 기반 융복합 의료기기 전문기업으로, 대표 제품 'CharmRenal'은 생분해성 고분자 기반의 생체적합 고성능 스캐폴드를 활용한 이식형 의료기기이다. 신장 조직의 재생을 유도해 기능을 회복시키는 혁신적 구조를 갖추고 있으며, 소동물실험에서 신장 기능이 15% 수준으로 저하된 상태에서 30~50%까지 회복하는 결과를 달성했다. 이는 사람의 경우 일상생활이 충분히 가능한 수준이다.

## 투자/성과
- 젠엑시스로부터 투자 유치 (금액 비공개)
- 딥테크 팁스 패스트트랙 선정, 3년간 15억 원 정부 R&D 자금 지원
- 소동물실험에서 신장 기능 15% -> 30~50% 회복 실증
- 향후 계획: 반려동물용 전임상 및 상용화, 인체 적용 임상시험 준비, GLP 기반 전임상/GMP 제조기술 개발, FDA/MDR/CE 인허가 전략 수립

## 딥테크 포인트
생분해성 고분자 기반 스캐폴드로 손상된 신장 조직의 재생을 유도하는 기술은 재생의학의 최전선에 위치한다. 현재 말기 신부전 환자에게는 투석이나 신장 이식 외에는 대안이 없는 상황에서, 스캐폴드를 통한 신장 기능 회복은 근본적인 치료 패러다임의 전환을 의미한다. 생체적합성/생분해성/조직 재생 유도 기능을 동시에 충족하는 소재 기술과 이를 의료기기로 구현하는 공학적 난제가 딥테크의 핵심이다.
