# 뉴빌리티 (Neubility)

> **분야**: 모빌리티 | **핵심기술**: 카메라 기반 자율주행 도심 라스트마일 물류 플랫폼

## 1. 기업 개요

- **설립연도**: 2017년 10월 26일
- **대표자**: 이상민 (Andrew Lee)
- **소재지**: 서울특별시 성동구 왕십리로 115, 2층 (성수동1가, 헤이그라운드)
- **홈페이지**: https://www.neubility.co.kr
- **주요 사업**: 카메라 기반 자율주행 배달/순찰 로봇 개발 및 RaaS(Robot-as-a-Service) 플랫폼 운영
- **임직원 수**: 약 64명 (2025년 12월 기준)
- **업력**: 8.3년
- **상태**: 비상장 (IPO 추진 중, NH투자증권 주관)

### 창업 스토리

이상민 대표는 연세대학교 천문우주학/기계공학 전공으로, 고등학교 시절 NASA 주관 대회에서 항공우주 부문 대상을 수상하고 대학에서 초소형 위성 및 위성 영상처리를 개발한 바 있습니다. 만 19세(2017년)에 동아리 친구 2명과 함께 뉴빌리티를 창업했습니다.

"처음엔 뚜렷한 목표 없이 공학적 지식을 바탕으로 무언가 만드는 일이 재미있어서 친구 둘과 함께 의기투합해 창업했다"며, 여러 시행착오를 겪다 자율주행 소프트웨어에 강점을 발견해 로봇 개발을 시작했습니다. 2021년 포브스(Forbes)지 선정 아시아 30세 이하 영 리더 30인에 선정되었습니다.

## 2. 핵심 기술 및 기술 차별성

### 카메라 기반 자율주행 (LiDAR-free)

뉴빌리티의 가장 큰 기술적 차별점은 **라이다(LiDAR) 없이 카메라만으로 자율주행**을 구현한다는 것입니다. 이는 하드웨어 비용을 대폭 절감하면서도 복잡한 도심 환경에서 안정적인 주행을 가능하게 합니다.

- **Multi-Camera V-SLAM**: 다중 카메라 기반 시각적 동시 위치추정 및 지도작성 기술
- **AI 기반 장애물 인식/회피**: 센서 퓨전을 통한 AI 기반 장애물 인식 및 회피 로직
- **주행 조건 인식/주행 가능 경로 탐지**: 실시간 환경 인식 및 경로 계획
- **Physical AI**: 로봇이 독립적으로 상황을 인지, 판단, 대응할 수 있는 피지컬 AI 역량

### 자율주행 데이터 축적

- 누적 자율주행 데이터: 10TB 이상
- 누적 주행거리: 78,497km (2025년 기준)
- 운영 사이트: 142개 (전국)
- 운영 로봇: 305대
- 연간 서비스 수행: 44,638회

### 기술 로드맵

뉴빌리티는 배달로봇에서 순찰로봇으로 확장하고 있으며, 향후 **휴머노이드 로봇** 기술과 **범용 로봇 파운데이션 모델** 개발을 목표로 하고 있습니다.

## 3. 제품/서비스

### Neubie (뉴비) - 자율주행 로봇

- 성인 무릎 높이의 소형 자율주행 로봇
- 3km 이내 단거리 이동 및 임무 수행
- 실내외 환경 모두에서 운영 가능
- 카메라 기반 자율주행으로 복잡한 도심 환경 대응

### Neubility Sense

- 소형 자율주행 모듈
- 기존 로봇에 통합하여 자율주행 기능 부여
- 서드파티 로봇과의 호환성 제공

### RaaS API (Robot-as-a-Service)

- 로봇 서비스 통합 API 플랫폼
- 비즈니스 파트너사 시스템과의 연동
- 로봇 관리, 모니터링, 주문 연동 기능

### 주요 서비스 영역

1. **배달 서비스**: 음식/소포 라스트마일 배달
2. **순찰 서비스**: 도심 환경 보안 순찰 로봇
3. **물류 서비스**: 산업단지/캠퍼스 내 물류 운송

### 주요 운영 사이트

- 인천 송도 센트럴파크
- 판교 테크노밸리
- 서울 강남
- 성남 서현역 (순찰로봇)
- 대학 캠퍼스 다수
- 아파트 단지

## 4. 딥테크 선정 내용

- **선정 사업아이템**: 자율주행 모빌리티 플랫폼
- **주요 내용**: 도심 자율주행 라스트마일 물류 플랫폼
- **딥테크 요소 분석**:
  - **기술 심화도**: 카메라만으로 자율주행을 구현하는 고도화된 컴퓨터 비전 및 AI 기술
  - **기술 장벽**: 복잡한 도심 환경(보행자, 차량, 신호등 등)에서의 안전한 자율주행
  - **데이터 우위**: 78,000km 이상의 실주행 데이터 축적으로 학습 데이터 경쟁력
  - **하드웨어 혁신**: LiDAR 없는 저비용 고성능 자율주행 로봇
  - **12대 신산업 분야**: 모빌리티
