## I. 시장 현황 및 문제 정의

### 시장 규모
- **TAM**(전체시장): 0~7세 자녀 보유/키즈 전체 시장 약 **40조원** (~200만명)
- **SAM**(유효시장): 용품+의류+여행 시장 약 **10조원+**
- **SOM**(수익시장): 용품+의류 시장 약 **8조원**

### 핵심 문제
1. **정보 부족**: 부모들이 자녀 연령/발달 단계에 맞는 소비 정보를 찾기 어려움
2. **신뢰 부재**: 기존 커머스 플랫폼에서 육아 특화 큐레이션 및 개인화 추천이 없음
3. **분산된 경험**: 콘텐츠(정보)와 커머스(구매)가 분리되어 있어 이탈률 높음

### 경쟁 환경
| 경쟁사 | 특징 | MAU | 투자 | 한계 |
|--------|------|-----|------|------|
| 베이비빌리 | 임신일기, 육아용품 판매 | 15~16만 | - | 폐쇄몰 운영, 데이터 기반 추천 미제공 |
| 마미톡 | 임산부/육아맘 커뮤니티 + 유아용품 쇼핑몰 | 15~16만 | - | 폐쇄몰 운영, 데이터 기반 추천 미제공 |
| 무무즈 | 키즈&패밀리 라이프스타일 편집샵 (소호/중저가 온라인 브랜드 위주) | 7~8만 | 누적 60억+ | 카테고리/브랜드 단위 단순 분류 |
| 아이보리 | 산후조리원 신생아 실시간 영상서비스 | ~3만 | - | 산후조리원 체류기간에 한정, 데이터 기반 추천 미제공 |
| 쑥쑥찰칵 | 사진 자동정리, 육아일기, 성장영상, 가족공유 | 6~7만 | 누적 22억 | Utility성 서비스, 상품 추천/판매 기능 없음 |

### 맘맘의 차별점
- **콘텐츠 + 커머스 통합 플랫폼**: 육아 '소비 관심사' 기반 상품 정보 탐색 → 맥락 소비 가능 → 개인화 추천
- **데이터 기반 초개인화**: 자녀 성별/개월수/관심 키워드/제품군 등 활용한 상품 랭킹 및 리뷰 제공
- **경쟁우위**: 국내 Baby Tech 중 유일하게 '소비 관심사'에 초점, 데이터 기반 상품/콘텐츠 추천 시스템 구축에 높은 경쟁우위 보유

## II. 솔루션 및 핵심 기술

### 4-모듈 시스템 아키텍처

```
[A] 데이터 수집 → [B] 데이터 가공 → [C] 분석·학습 → [D] 추천 시스템
```

#### [A] 데이터 수집 모듈
- 사용자 행동 데이터 (클릭, 체류시간, 구매, 검색)
- 아이 프로필 데이터 (연령, 발달단계)
- 상품/콘텐츠 메타데이터

#### [B] 데이터 가공 모듈
- **YOLO v5 기반 이미지 분류**: 상품 이미지 자동 카테고리 분류 및 태깅
- **BERT 기반 리뷰 감성분석**: 사용자 리뷰에서 감성(긍정/부정/중립) 및 핵심 키워드 추출
- **Multi-label Classification**: 하나의 상품/콘텐츠에 다중 속성 태그 자동 부여

#### [C] 분석·학습 모듈
- **Meta-Prod2Vec 추천 엔진**:
  - 기존 추천 시스템의 **Cold Start 문제** 해결 (신규 사용자/상품에도 추천 가능)
  - **Popular Item Bias** 해소 (인기 상품 편중 방지)
  - 상품의 메타데이터(카테고리, 브랜드, 연령대)를 임베딩에 함께 학습
- **LSTM 시계열 예측**: 수요 예측, 사용자 생애주기 기반 소비 패턴 예측

#### [D] 추천 시스템 모듈
- 개인화 상품 추천 (홈피드, 카테고리별)
- 맞춤 콘텐츠 추천 (발달단계별 육아 정보)
- Push 알림 최적화 (개인별 최적 시간/내용)
- 검색 결과 개인화

### 핵심 성과지표 (KPI)

| 지표 | 목표 | 설명 |
|------|------|------|
| 육아 컨텐츠 추천 정확도 (NDCG@20) | ≥ 0.161 | 비중 40%, 벤치마크: Facebook 0.161 |
| 맞춤형 컨텐츠 푸시 알림 CTR | 8% | 비중 20%, 벤치마크: Android 4.1%/iOS 1.7% |
| 시계열 분석 모델 정확도 (RMSE) | < 20 | 비중 30%, 내부 컨텐츠 CTR 근거 |
| 서버 안정성 (Crash rate) | < 3% | 비중 10%, Google Play Console 측정, 벤치마크: Top 50% Percentile 1% |

> 모든 성과지표는 **TTA 한국정보통신기술협회 V&V 공인시험성적서**로 평가 (서면평가 보완 후 확정)

## III. 스케일업 전략

### 사업 목표 (연구개발 완료 후)
- 연간 거래액: 약 **2,000억원**
- MAU: 약 **100만명**
- 매출: 약 **300억원** (이 중 연구개발 결과물 관련 매출 약 200억원)

### 수익 모델
1. **판매 수수료**: 15~20% (카테고리별 차등)
2. **AI 기반 광고**: 개인화 추천 기반 타겟 광고 상품

### 글로벌 확장 전략
9개국(중국, 미국, 일본, 베트남, 말레이시아, 필리핀, 싱가포르, 인도네시아, 태국) 분석 후 우선 진출 대상:
- **미국**: 재미 한인 시장(250만+) 우선 공략, 프리미엄 제품군 수요
- **베트남**: 동남아 시장 교두보, 높은 출생률
- **인도네시아**: 인구 대국, 성장하는 육아 시장

**로드맵**: 2024 국내 커머스 플랫폼 안정화 → 2025 해외 결제/배송 및 현지 DB 구축 → 2026~ 다국어 서비스 제공 및 글로벌 시장 확대

### 현재 성과 (계획서 작성 시점)
- MAU: 3만명+, DAU: 3천+
- WAU/MAU Ratio: 약 50% (높은 고착도)
- M+1 리텐션: 약 60%, M+5까지 30%+ 유지
- 콘텐츠 조회수: 200만+
- 주요 사용자층: 0~7세 자녀를 둔 부모
- 커머스 타임딜 테스트('21.11): 37% 구매전환, 3시간 거래액 약 1,500만원
